# Таблица с внутренним скроллом и возможностью удаления объектов

### [Посмотреть проект в один клик](https://webbOrista.github.io/list-of-water-meters/)

## Стек

- **React**
- **TypeScript**
- **Mobx-state-tree**
- **HTML**
- **CSS**

## Дополнительно использовал библиотеки

- Axios - для удобства реализации API
- React-spinners - для аккуратного лоадера на время загрузки данных

## Структура

Проект разделен на 5 основных компонентов:

- **Header** — шапка приложения с заголовком
- **Table** — общий компонент таблицы
- **TableRow** — отдельная строка таблицы
- **DeleteButton** — кнопка удаления счетчика
- **Pagination** — кнопки пагинации

## Особенности

- Использовал подход styled-components для стилизации приложения
